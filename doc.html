Framework CSS Moderno

Un framework CSS ligero, personalizable y sin dependencias. Perfecto para crear interfaces modernas y responsivas.

⚡ Rápido

Sin JavaScript, sin frameworks pesados. Solo CSS puro y optimizado.

🎨 Personalizable

Variables CSS para cambiar colores, espaciado y tipografía fácilmente.

📱 Responsivo

Grid y componentes que se adaptan automáticamente a cualquier pantalla.

🌙 Dark Mode

Soporte automático para modo oscuro basado en preferencias del sistema.

🎯 Minimalista

Solo lo necesario. Sin bloat, sin dependencias externas innecesarias.

♿ Accesible

Componentes construidos con buenas prácticas de accesibilidad.

Instalación

Comenzar con el framework es muy simple. Solo necesitas descargar el archivo CSS e incluirlo en tu HTML.

Paso 1: Descarga el archivo

Descarga framework.css de la carpeta del proyecto.

Paso 2: Importa en tu HTML

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mi Página</title> <link rel="stylesheet" href="framework.css"> </head> <body> <!-- Tu contenido aquí --> </body> </html>

Paso 3: Comienza a usar las clases

<button class="btn btn-primary">Mi botón</button> <div class="card"> <div class="card-header"> <h2>Título</h2> </div> <div class="card-body"> Contenido de la tarjeta </div> </div>

Alertas

Componentes para mostrar mensajes importantes al usuario.

Tipos de Alertas

ℹ
Información: Este es un mensaje informativo.
✓
Éxito: La operación se completó correctamente.
⚠
Advertencia: Revisa esta información importante.
✕
Error: Algo salió mal. Intenta de nuevo.

Código

<div class="alert alert-success"> <span>✓</span> <div> <strong>Éxito:</strong> Tu acción fue completada. </div> </div>
